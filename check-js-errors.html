🛠️ Manual Error Checking

  1. Open the dashboard in a new tab:
  2. Open browser console (F12 → Console tab)
  3. Look for red error messages
  4. Try clicking timer buttons and watch for errors
  5. Common errors to look for:
    • "Uncaught TypeError" - function not found
    • "Uncaught ReferenceError" - variable not defined
    • "Uncaught SyntaxError" - code syntax issue
